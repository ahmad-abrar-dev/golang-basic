========SWITCH TO USE GO ROOT / GO MOD========
to use the GOROOT make sure to inactive write this line of command on the terminal
> go env -w GO111MODULE=off

NOTE: But after this command, you will be not able to use go mod init through terminal. You must create "go.mod" file manually.

to use somthing like gomod feature make sure to set on the GO111MODULE
>  go env -w GO111MODULE=on
